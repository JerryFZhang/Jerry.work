IndoorNav

IndoorNav is an indoor navigation solution startup based in Ottawa. Built by Abdul and me, our positioning algorithm provides you by far the most accurate indoor mapping technology, with incredibly low energy consumption.

The GreatMovieDatabase is a web application that pulls movie information out from a PostgreSQL database. Built using Node.js, AngularJS, Express.js and Bootstrap. Database insertion code generated by Python. Open sourced( App, Database)
